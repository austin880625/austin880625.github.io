關於我和這個網站

GitHub

自己嘗試寫了一個 static site generator ，因為覺得自己文章量不多且未來頂多把文章的 markdown 檔案搬去別的 SSG 就好，就把這個網站直接上線看看，缺的功能就慢慢一邊寫一邊補上。

受到 vanilla.js 及 MotherFuckingWebsite 的啟發，希望這個網站能在內容以外的東西盡量簡單，讓我能盡量偷懶。也所以這個網站目前簡單到能用 w3m 瀏覽器正常顯示。
